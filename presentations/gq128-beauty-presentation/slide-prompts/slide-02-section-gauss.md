# Slide 02 — Эпиграф Гаусса и таймлайн

> Читать сначала: [`_shared.md`](./_shared.md), затем `../BRIEF.ru.md` §§ 4, 7 (строка слайда 2), §14 (цитаты).

## Роль в арке доклада

Первая section-заставка. Ставит **конституцию доклада**: «технический
долг» как сюжетная метафора. Гаусс 1831 года жалуется, что $+1$, $-1$,
$i$ назвали «позитивным, негативным, мнимым» вместо «прямого, обратного,
бокового», — и эту жалобу доклад принимает всерьёз.

Этот слайд также содержит **манифест результата**: на таймлайне стоит
«1959, Сетунь» и оборот на иерархию упаковок шаров, чтобы дальше
маршрут 3 → 14 был уже предчувствован.

## Длительность и «кончик скотча»

- На сцене: 60–90 секунд.
- **Кончик скотча:** одна цитата Гаусса крупным шрифтом и старая дата
  «1831» рядом. Жалоба великого человека, с которым трудно спорить. Пока
  зритель читает цитату, под ней **медленно** проявляется таймлайн.

## Content

### Визуал

Двухколоночный кадр (в presentation mode; в reader mode складывается):

- **Слева:** эпиграф (цитата) крупным шрифтом, автор + дата снизу
  одной строкой. Это **центральный якорь** слайда.
- **Справа / снизу:** интерактивный горизонтальный таймлайн с четырьмя
  узлами: **1748**, **1799**, **1831**, **1959**. Линия — это тонкая
  горизонталь цвета `var(--class-muted)`. Узлы — круги `var(--class-primary)`.
  Над каждым узлом всплывает короткая подпись по hover/tap.

### Интерактив

Одна директива:

```
:::epigraph-timeline
:::
```

### Эпиграф (дословный текст)

Берётся из BRIEF §14:

> *«Had +1, −1, and i been named direct, inverse, and lateral rather
> than positive, negative, and imaginary, the darkness would never have
> formed.»*

Под цитатой — строка-атрибуция:

> — К. Ф. Гаусс, 1831

### Подпись (одна строка)

Под таймлайном:

> *Это — один абзац из письма Гаусса 1831 года. Мы возьмём его всерьёз.*

## Компоненты для реализации

### `EpigraphTimeline`

**Назначение.** Статичная цитата слева + интерактивный таймлайн справа.
Компонент включает обе части (а не только таймлайн), потому что они
жёстко согласованы по вертикали — цитата должна «обнимать» таймлайн.

**Файл.** `presentations/gq128-beauty-presentation/components/EpigraphTimeline.tsx`.

**Директива.** `:::epigraph-timeline`.

**Props.**

```ts
type EpigraphTimelineProps = Record<string, never>;
```

Все данные зашиты — таймлайн одной презентации, не переиспользуется.

**Данные таймлайна** (зашиты в компоненте):

```ts
const events = [
  {
    year: 1748,
    title: "Эйлер",
    tooltip: "«Introductio in analysin infinitorum». Первое систематическое использование i и e^iφ.",
  },
  {
    year: 1799,
    title: "Вессель",
    tooltip: "«On the Analytical Representation of Direction». Первая геометрическая интерпретация комплексных чисел. Оставалась незамеченной 98 лет.",
  },
  {
    year: 1831,
    title: "Гаусс",
    tooltip: "Предлагает переименовать +1 / −1 / i в direct / inverse / lateral. Его не послушали.",
  },
  {
    year: 1959,
    title: "Сетунь",
    tooltip: "Брусенцов, МГУ. Троичная ЭВМ со сбалансированным кодом {−1, 0, +1}. Обобщим на комплексную плоскость.",
  },
];
```

**Визуал.**

- Горизонтальная линия длиной 100% ширины контейнера.
- Четыре круга радиуса 10 px (в presentation mode — 14 px) по оси.
- Позиционирование пропорционально годам? **Нет** — равномерно, иначе
  1748–1799 станет неотличимо от 1799–1831. Дата подписывается под
  узлом `text-sm`.
- Над узлом «1831» **стоит постоянная стрелка** (треугольник) и
  микро-подпись «эпиграф» — это визуальная рифма между цитатой и
  таймлайном.
- Tooltip появляется по hover (desktop) и по tap (mobile).

**State.**

```ts
const [activeYear, setActiveYear] = useState<number | null>(null);
```

`null` — никто не наведён. На мобильном tap по узлу ставит active;
повторный tap по тому же узлу снимает active; tap снаружи — тоже
снимает.

**Анимация.**

- При монтировании: цитата проявляется (opacity 0 → 1, 400 мс).
- Через 500 мс после цитаты начинает «вырисовываться» линия таймлайна
  (stroke-dasharray, 800 мс).
- По мере прохождения линии появляются узлы, один за другим (stagger
  150 мс).
- Tooltip: translateY(4px → 0) + opacity, 180 мс.

**Reader mode.**

- Цитата сверху, таймлайн снизу.
- Tooltips срабатывают по tap.
- Линия таймлайна остаётся горизонтальной, но узлы крупнее (18 px) и
  дат чуть меньше.

**Reduced motion.**

- Никакого проявления. Цитата и таймлайн с самого начала в финальном
  виде. Tooltips по hover/tap остаются (это осознанный жест
  пользователя).

**A11y.**

- Блок цитаты — `<blockquote>` + `<cite>`.
- Таймлайн — `<ul role="list">`; каждый узел — `<li>` с
  `<button aria-describedby="tooltip-NNNN">`.
- Tooltip — `<div role="tooltip" id="tooltip-NNNN">`.
- Стрелки ←/→ перемещают фокус между узлами.

## Токены и стили

  слов в русском переводе.
- Атрибуция: Open Sans 400, кегль 16 px, `color: var(--class-primary)`.
- Таймлайн: линия `stroke: var(--class-muted)`, узлы `fill:
  var(--class-primary)`, активный узел `fill: var(--class-accent)`.
- Tooltip фон: `var(--class-muted)`; рамка: `var(--class-primary)`;
  текст: `var(--class-foreground)`.

## Проверка «кончик скотча» и «слайд с середины»

- **Кончик скотча:** первая реплика докладчика — чтение цитаты вслух
  на английском (или русском — по выбору). Она **уже** есть на сцене,
  не нужна подводка.
- **Слайд с середины:** цитата + таймлайн читаются как один кадр;
  интерактив (tooltips) дополняет, но не критичен. Если зритель попал
  сюда на 40-й секунде, он видит всё ключевое сразу.

## Что НЕ делать

- Не вставлять портрет Гаусса. Портрет переводит внимание с цитаты на
  бюст.
- Не пропорционализировать годы по шкале. 211 лет между 1748 и 1959
  никогда не поместятся красиво — равномерная шкала информативнее.
- Не ставить на узел 1959 никакого текста про GQ128. Узел 1959 говорит
  «Сетунь», и всё. Мост на GQ128 делается голосом или в бэкстейдже.
