# Slide 03 — Пифагорейская катастрофа

> Читать сначала: [`_shared.md`](./_shared.md), затем `../BRIEF.ru.md` §7 (строка слайда 3).

## Роль в арке доклада

Первый «исторический» слайд. Готовит почву для трёх последующих
исторических слайдов (Евдокс / Декарт / комплексные числа),
показывая, что **числа и геометрия поссорились задолго до $i$**. У
пифагорейцев была простая и красивая картина мира, где все отношения
длин — рациональны. Один треугольник её сломал.

Ни одной формулы на сцене. Всё — через две картинки, которые разговаривают
друг с другом.

## Длительность и «кончик скотча»

- На сцене: 60–75 секунд.
- **Кончик скотча:** зритель видит **два треугольника** бок о бок —
  обычный (3, 4, 5) и прямоугольный равнобедренный (1, 1, ?). Вопрос
  «чему равна гипотенуза второго?» настолько общеизвестен, что **жест
  клика по гипотенузе** становится самопровозглашающимся.

## Content

### Визуал

Два прямоугольных треугольника, каждый на своей стороне сцены:

- **Слева:** треугольник (3, 4, 5). Катеты подписаны «3» и «4», гипотенуза
  подписана «5». Все подписи видны **сразу**.
- **Справа:** треугольник (1, 1, ?). Катеты подписаны «1» и «1».
  Гипотенуза подписана знаком «?» **до клика**.

При клике по гипотенузе правого треугольника:

1. Появляется надпись: «$\sqrt{2}$».
2. Под надписью раскрывается полоса десятичного разложения:
   `1.41421356...`, печатаясь по одной цифре (тайпвритер, 60–80 мс
   на цифру, 18–22 цифры всего, потом «…»).
3. Под разложением мягко проявляется подпись: «Это число не имеет конца
   и не повторяется».

Повторный клик по гипотенузе **скрывает** разложение и возвращает знак
«?», чтобы докладчик мог повторить жест для другой подгруппы зрителей.

### Подпись (одна строка)

Под двумя треугольниками, после раскрытия:

## Backstage

Разделы:

1. **Доказательство иррациональности $\sqrt 2$.** Классическое
   от противного: $\sqrt 2 = p/q$, предположим $p, q$ взаимно просты, тогда
   $p^2 = 2q^2 \Rightarrow p$ чётное $\Rightarrow p = 2k \Rightarrow
   4k^2 = 2q^2 \Rightarrow q^2 = 2k^2 \Rightarrow q$ чётное — противоречие.
   Формулы в KaTeX, разбираем за 6 строк.
2. **Гиппас и пифагорейцы.** Легенда об ученике, «слившем» миру
   существование несоизмеримых. Историческое значение: первый кризис
   оснований математики.
3. **Мост к float.** В IEEE 754 у $\sqrt 2$ тоже нет точного
   представления. Любая программа, делающая
   `Math.sqrt(2) * Math.sqrt(2) !== 2` — прямой наследник этой
   катастрофы. Короткий сниппет JS:
   ```js
   > Math.sqrt(2) * Math.sqrt(2)
   2.0000000000000004
   ```
4. **Как пифагорейцы справились.** Перерос в следующий слайд (Евдокс):
   если чисел не хватает — откажемся от чисел в пользу геометрии.

## Компоненты для реализации

### `PythagoreanTriangles`

**Файл.** `presentations/gq128-beauty-presentation/components/PythagoreanTriangles.tsx`.

**Директива.** `:::pythagorean-triangles`.

**Props.**

```ts
type PythagoreanTrianglesProps = {
  digits?: number;  // сколько цифр разложения √2 печатать. Default: 20.
  typingSpeedMs?: number;  // скорость тайпвритера. Default: 70.
};
```

**Визуал.**

- Вся сцена — один SVG с двумя треугольниками, центрированными в своих
  половинах.
- Треугольники **одинаковой визуальной высоты** (независимо от того, что
  высота первого 4, а второго 1) — это важно: зритель не должен читать
  пропорции как «они разной высоты, поэтому и гипотенузы разные». Шкала
  подбирается так, что самый длинный катет обоих занимает одинаковое
  экранное расстояние.
- Линии треугольников: `stroke: var(--class-primary)`, `stroke-width: 2`.
- Гипотенуза, пока на ней стоит «?», чуть толще (3) и цвета
  `var(--class-accent)` — подсказка, что её нужно потыкать.
- Цифры подписей — Open Sans 500, 16 px; знак «?» — 22 px, жирнее.
- Под треугольниками — полоса для десятичного разложения, монопростарный
  шрифт (`JetBrains Mono` если уже есть; иначе `ui-monospace, monospace`),
  кегль 20 px.

**State.**

```ts
const [revealed, setRevealed] = useState(false);
const [typed, setTyped] = useState<string>(""); // постепенно растёт
```

**Анимация.**

- Клик/tap по правой гипотенузе: `revealed = true`, запускается
  тайпвритер через `setInterval` с шагом `typingSpeedMs`.
- Клик по правой гипотенузе повторно: `revealed = false`, `typed = ""`.
- Сам треугольник не анимируется — это тихая статика, чтобы разложение
  было главным событием.

**Точность разложения.**

Цифры $\sqrt 2$ — **захардкоженная строка** в компоненте:
```
1.41421356237309504880168872420969807856967187537694807317667973799...
```

Никаких runtime-вычислений (JS `Math.sqrt(2)` даёт только 17 цифр точно;
нам нужно больше, и это простая строка).

**Reader mode.** Треугольники складываются вертикально, подпись остаётся
ниже. Кликабельная гипотенуза получает чуть больший hit-area (≥ 44×44 px).

**Reduced motion.** Тайпвритер заменяется мгновенным показом всей строки
разложения.

**A11y.**

- Правая гипотенуза — `<g role="button" tabindex="0" aria-label="Показать
  длину гипотенузы равнобедренного прямоугольного треугольника с катетами 1
  и 1">`, реагирует на Enter/Space.
- При раскрытии: `aria-live="polite"` вокруг блока с разложением, чтобы
  screen reader зачитал «квадратный корень из двух, приблизительно
  один целый четыреста один…».

## Проверка «кончик скотча» и «слайд с середины»

- **Кончик скотча:** зритель узнаёт задачу до того, как докладчик
  заговорит. Это детская задача, которую все помнят.
- **Слайд с середины:** если зайти в момент активной печати цифр — всё
  равно читается: большой `\sqrt 2` + лента цифр + подпись «не имеет
  конца».

## Что НЕ делать

- Не рисовать знак $\sqrt{}$ на гипотенузе до клика. Это spoiler.
- Не давать численной пропорциональности (левый треугольник не должен
  быть в 4 раза больше правого). Главный сюжет — о числах, а не о
  масштабе.
- Не показывать доказательство иррациональности на сцене. Только в
  бэкстейдже.
