subtitle: "A New Approach"
---

Each slide is a separate web page with its own unique URL. Participants can follow the presentation on their own devices, interact with the content, and revisit the materials at any time.

:::callout{tone=accent}
Any slide can be extended with an interactive React component via a `:::name{...}` directive. On GitHub the directive shows as plain text; in the browser it renders as a live block.
:::
