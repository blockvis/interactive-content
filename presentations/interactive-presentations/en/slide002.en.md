Every slide is a standalone web page with a unique URL. Attendees can follow the presentation on their own devices, interact with the content, and revisit the materials at any time.

:::callout{tone=accent}
Any slide can be extended with an interactive React component via the `:::name{...}` directive. On GitHub the directive renders as text; in the browser — as a live block.
:::
