Cada diapositiva es una página web independiente con una dirección única. Los participantes pueden seguir la presentación en sus propios dispositivos, interactuar con el contenido y volver a los materiales en cualquier momento.

:::callout{tone=accent}
Cualquier diapositiva puede ampliarse con un componente React interactivo mediante la directiva `:::nombre{...}`. En GitHub la directiva se muestra como texto; en el navegador, como un bloque vivo.
:::
