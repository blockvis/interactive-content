Chaque diapositive est une page web distincte avec une adresse unique. Les participants peuvent suivre la présentation sur leurs appareils, interagir avec le contenu et revenir aux supports à tout moment.

:::callout{tone=accent}
Toute diapositive peut être enrichie d'un composant React interactif via la directive `:::nom{...}`. Sur GitHub, la directive apparaît comme du texte ; dans le navigateur, comme un bloc vivant.
:::
